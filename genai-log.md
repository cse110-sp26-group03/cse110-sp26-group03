# GenAI Usage Log

This document tracks significant usage of Generative AI tools during development.

The purpose of this log is transparency and documentation of how AI-assisted work was incorporated into the project.

Minor usage (small syntax questions, autocomplete, etc.) does not necessarily need to be logged. Significant design, implementation, documentation, or debugging assistance should be recorded.

---

# Log Entry Template

Copy and paste this section for each new entry.

```md
## Date: YYYY-MM-DD

### Team Member(s)
- Name

### Tool Used
- ChatGPT / Claude / GitHub Copilot / Cursor / etc.

### Purpose
Briefly describe what the AI tool was used for.

### Prompt Summary
Summarize the prompts or requests given to the AI tool.

### Output Used
Describe what portions of the output were actually incorporated into the project.

### Human Modifications / Review
Describe any edits, verification, debugging, refactoring, or review performed by team members before using the output.

### Related Files / Issues (Optional)
- `/src/example.js`
- `Issue #12`
```
